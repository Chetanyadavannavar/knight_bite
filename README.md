# 🍔 Knight Bite Clone

A responsive front-end clone of the Knight Bite website built using **HTML5, CSS3, and JavaScript**. This project recreates the modern user interface of the original website while showcasing responsive web design and front-end development skills.

## 🚀 Live Demo

🔗 https://chetanyadavannavar.github.io/knight_bite/

> *(Update this link after deploying the project.)*

---

## 📌 Features

- 🍔 Responsive and modern landing page
- 📱 Mobile-friendly design
- 🎥 Background video on the home page
- 🧭 Navigation bar with multiple pages
- 📋 Interactive menu section
- 🏪 Franchise information page
- 📞 Contact and location section
- 🎨 Clean and attractive UI
- ⚡ Fast loading and lightweight

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Chetanyadavannavar/knight_bite.git
```

Move into the project folder:

```bash
cd knight_bite
```

Open the project by launching `index.html` in your browser.
